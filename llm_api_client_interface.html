Abstract base class for Language Model API client interfaces.

This class defines the interface for interacting with a Language Model API. Concrete implementations must inherit from this class and provide implementations for the abstract methods.

Attributes:

None

Methods:

To create a custom API client for a specific language model, inherit from this class and provide implementations for the abstract methods.
